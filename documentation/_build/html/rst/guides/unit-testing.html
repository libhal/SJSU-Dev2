2.2. Unit Testing¶

2.2.1. Unit Testing Tools¶

SJTwo uses Catch2, Fakeit and FFF (fake function framework).

2.2.2. Compiling and Running Tests¶

Make sure you are in a project directory and run make test to compile, run, evaluate using valgrind and produce code coverage for your project and the library files.

2.2.3. Created additional unit tests¶

Simply end the name of your file <insert title here>_test.cpp and it will be included in the code testing. Do not name a file that is not meant to be a test _test.cpp.

Error

DO NOT IMPLEMENT a main function anywhere in your tests. This will cause linking issues since main is already defined in the library/L5_Testing/main_test.cpp file.
