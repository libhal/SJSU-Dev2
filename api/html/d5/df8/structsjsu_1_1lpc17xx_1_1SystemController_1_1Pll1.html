The available PLL1 multipliers.

Note
Since only the divider value of 2 is available for PLL1, the multiplier value should be selected based on the main oscillator.
See also
Table 37. PLL1 Multiplier values https://www.nxp.com/docs/en/user-guide/UM10360.pdf#page=55
