◆ Calibrate()

void sjsu::StopWatch::Calibrate ( )
inline

Calibrates the stopwatch by figuring out how much time it takes to Start() and Stop() the stopwatch. Since it takes time to call the methods, this is used to remove the call to Stop() and Start() from the time between events, giving a more accurate time.
