template<AddressWidth address_width, std::endian endianness>
struct sjsu::MemoryAccessProtocol::Specification_t< address_width, endianness >

Specifies information about the device's memory

Template Parameters
Defineshow many bytes are used to address memory. Typically for I2C devices they use 8-bits (or 1 byte) for memory addresses. Some devices have 32-bit address spaces thus they need 4 bytes.
Definesthe endianess of the system. Allows the device memory map to swap the endianness of whole integer values when they do not match the system's endianness.
