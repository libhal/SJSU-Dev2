Detailed Description

template<size_t kNumberOfCommands>
struct sjsu::CommandList_t< kNumberOfCommands >

In order to construct a CommandLine object, you must provide a command list structure with static storage duration (must be a global variable).

See also
{https://en.cppreference.com/w/cpp/language/storage_duration}
