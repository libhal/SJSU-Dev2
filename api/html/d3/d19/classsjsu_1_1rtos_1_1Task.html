Detailed Description

template<size_t kStackSize>
class sjsu::rtos::Task< kStackSize >

An abstraction layer for FreeRTOS tasks. All tasks must inherit this interface and override the Run() function.

Attention
All tasks must be persistent or in global space.
Template Parameters
kStackSizeThe pre-allocated stack size of this task in bytes.

inlineoverridevirtual

Sets the delay time to ensure Run() is called at a desired frequency for periodic tasks.

For examples, if time = 1000 and the sjsu::rtos tick period = 1ms, then Run() will be called every 1 second.

Parameters
